## tplinkIoT

Scripts to interact with tplink IoT devices. Brief description of the existing scrips:

`tplink_discover.py`: Get IP of devices that respond to the discovery process of the TP-LINK Smart home protocol. 

`cam_loc_stream.sh`: get local stream from IP cameras. This script has been only tested with the `EC60(US)` (aka Kasa Spot). You do need to get the auth token (as in [this article](https://hkubota.wordpress.com/2022/05/03/tp-link-kasa-kc120-streaming-without-kasa/)).
